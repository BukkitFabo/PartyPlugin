PartyPlugin
===================
Ein simples PartyPlugin, welches auf keinem MiniGames-Server fehlen sollte.

Commands:
---------------

	* /party invite <Spieler> - Lädt einen Spieler in deine Party ein.
	* /party accept <Spieler> - Akzeptiert die Partyeinladung von einem Spieler.
	* /party deny <Spieler> - Lehnt die Partyeinladung von einem Spieler ab.
	* /party kick <Spieler> §f- Kick einen Spieler aus deiner Party.
	* /party msg <Nachricht> §f- Sendet an alle Spieler in der Party eine Nachricht.
	* /party promote <Spieler> §f- Setzt einen neuen PartyOwner.
	* /party delete - Löst die existierende Party auf.
	* /party leave - Verlässt die Party, in der du bist.
	* /party list - Zeigt dir alle Spieler die in der Party sind.
	* /party toggle - Aktiviert oder Deaktiviert deine Partyanfragen.
	
	
Config:
---------------

	* MySQL.yml zum einstellen der Datenbankdaten(Host, Database, User, Password)
